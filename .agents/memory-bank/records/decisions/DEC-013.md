# DEC-013 — Externalized SQL + named bind parameters

| Field | Value |
|---|---|
| Status | Accepted |
| Date | 2026-06-17 |
| Modules | `core`, `chat`, any JDBC repository |
| Superseded by | — |

## Rationale

Reviewable SQL, no string literals in Java, safe parameter binding; proven in
[ai-architect-6-mcp](https://github.com/berdachuk/ai-architect-6-mcp/blob/main/src/main/java/com/example/medicalmcp/medicalcase/repository/impl/MedicalCaseRepositoryImpl.java).

## Consequence

`@InjectSql` + files under `src/main/resources/sql/`; `NamedParameterJdbcTemplate` with `:name` binds only; no inline SQL, no `?` positional JDBC in app code.
